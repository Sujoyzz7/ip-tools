API Endpoints

IP Lookup

GET https://api.iplookup.com/v1/ip/{ip_address}
Parameters
Parameter Type Required Description
ip_address string Yes The IP address to lookup
Example Response

  "ip": "8.8.8.8",
  "country": "United States",
  "country_code": "US",
  "region": "California",

  "asn": "AS15169 Google LLC",
  "proxy": false,
  "hosting": true
}

Authentication

All API requests must include your API key in the header:

X-API-Key: your_api_key_here

Getting Your API Key

  1. Sign up for a free account
  2. Navigate to your dashboard
  3. Generate an API key
  4. Include the key in your requests

Error Handling

Status Code Meaning Description
200 Success Request completed successfully
400 Bad Request Invalid IP address or parameters
401 Unauthorized Missing or invalid API key
429 Rate Limited Too many requests
500 Server Error Internal server error

Code Examples

JavaScript

const apiKey = 'your_api_key';
const ip = '8.8.8.8';

fetch(`https://api.iplookup.com/v1/ip/${ip}`, {
  method: 'GET',
  headers: {
    'X-API-Key': apiKey,
    'Content-Type': 'application/json'
  }
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));

Python

import requests

api_key = 'your_api_key'
ip = '8.8.8.8'

url = f'https://api.iplookup.com/v1/ip/{ip}'
headers = {
    'X-API-Key': api_key,
    'Content-Type': 'application/json'
}

response = requests.get(url, headers=headers)
data = response.json()
print(data)

cURL

curl -X GET \
  https://api.iplookup.com/v1/ip/8.8.8.8 \
  -H 'X-API-Key: your_api_key' \
  -H 'Content-Type: application/json'
